## Project Overview
ImpressiveGift is a frontend UI development project designed for an e-commerce–style gift platform.  
The goal of this project is to create a clean, responsive, and user-friendly interface that visually represents the flow of an online gift shopping application.

During this internship, my role focused on:
- Designing user interface layouts
- Developing reusable React components
- Structuring the project using best practices
- Implementing responsive UI elements

## How to Run the Project (Vite)

2. Start the development server:

3. Open your browser and navigate to:
